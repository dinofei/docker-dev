API名称. 如果没有特殊, 默认为 请求域名 或者 请求路径.

如果DNS地址指向你的API地址. 譬如, mockbin.com. 至少 请求域名 或者 请求路径 或者两者都指向.

如果路径地址指向你的API地址. 譬如, /someservice. 至少 请求域名 或者 请求路径 或者两者都指向.

请求路径目录在代理的最终请求API之前. 譬如需要请求 /someservice/hello 将被解析成 upstream_url/hello. 默认是否.

保留原 Host 主机头发送到客户端, 替换为 hostname 主机名 upstream_url. 默认是否.

基础的目的地URL指向你的API服务器, URL将采用代理请求转发. 譬如, https://mockbin.com.
